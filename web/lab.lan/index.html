Installing the CA Certificate

Download skypaw.crt using the button above, then follow the steps for your OS. You only need to do this once — it covers all services.

🐧 Arch Linux

sudo cp skypaw.crt /etc/ca-certificates/trust-source/anchors/
sudo update-ca-trust

🐧 Ubuntu / Debian

sudo cp skypaw.crt /usr/local/share/ca-certificates/
sudo update-ca-certificates

🍓 Raspberry Pi OS

sudo cp skypaw.crt /usr/local/share/ca-certificates/
sudo update-ca-certificates

🎩 Fedora / RHEL

sudo cp skypaw.crt /etc/pki/ca-trust/source/anchors/
sudo update-ca-trust extract

❄️ NixOS

# configuration.nix
security.pki.certificateFiles = [
  /path/to/skypaw.crt
];

Then sudo nixos-rebuild switch

🦊 Firefox

Firefox uses its own certificate store independent of the OS.

  1. Open Settings → Privacy & Security
  2. Scroll to Certificates → click View Certificates…
  3. Go to the Authorities tab → click Import…
  4. Select skypaw.crt
  5. Check "Trust this CA to identify websites" → OK

Verify with: curl https://adguard.lan — should return a response, not an SSL error.
